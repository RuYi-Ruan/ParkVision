# ParkVision

毕业设计项目骨架，采用前后端分离与算法模块独立组织：

- `backend/`：Django + DRF 后端
- `frontend/`：Vue + Vite 前端
- `ai/`：检测、识别与推理服务
- `data/`：初始化数据、样例数据与备份
- `docs/`：需求、设计、测试与论文材料
- `deploy/`：部署配置
- `scripts/`：辅助脚本
- `logs/`：运行日志
- `tests/`：项目级测试

当前阶段已完成目录初始化，后续可继续执行：

1. Django 项目与各 app 初始化
2. Vue 项目初始化与路由/状态管理接入
3. AI 推理接口与后端联调骨架搭建
